# Step 4: Using Pomodazzle

## 4.1 Start the timer
- Click the timer to start the countdown.
- Study
- Hear the sound for the break after the work time has elapsed.

## 4.2  Take Breaks
- When the Pomodoro timer finishes, Pomodazzle will notify you to take a 5-minute break.
- After your break, you will hear the sound again to start the next Pomodoro session.
